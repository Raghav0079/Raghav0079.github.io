# 🚀 Raghav Mishra - 3D Interactive Portfolio

A modern, fully responsive 3D interactive portfolio showcasing my work as an AI researcher and developer, built with Three.js and cutting-edge web technologies.

🌐 **Live Demo**: [https://raghav0079.github.io](https://raghav0079.github.io)

## ✨ Features

- **🎮 3D Interactive Elements**: Powered by Three.js with floating geometric shapes and animations
- **🎨 Modern Design**: Glassmorphism UI, gradient backgrounds, and smooth animations  
- **📱 Fully Responsive**: Optimized for all devices - mobile, tablet, and desktop
- **⚡ Fast Loading**: Optimized performance with efficient 3D rendering
- **🔍 SEO Optimized**: Complete meta tags for search engines and social sharing

## 🎯 Pages

- **🏠 Home** (`3d-portfolio.html`) - Interactive landing page with 3D elements
- **👨‍💻 About** (`about.html`) - Skills, tools, and background with neural network visualization  
- **🏆 Certifications** (`certifications.html`) - Professional certificates and achievements
- **🚀 Projects** (`projects.html`) - Showcase of AI/ML projects and innovations
- **📞 Contact** (`contact.html`) - Contact form and social media links

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript
- **3D Graphics**: Three.js (r128)
- **Styling**: Custom CSS with CSS Grid, Flexbox, and CSS Animations
- **Hosting**: GitHub Pages
- **CDN**: Cloudflare for Three.js library

## 🚀 Deployment

The portfolio is automatically deployed via GitHub Pages. Any changes pushed to the `main` branch will be reflected live.

**Deployment URL**: [https://raghav0079.github.io](https://raghav0079.github.io)

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the `portfolio-project` directory.
3. Open `src/index.html` in your web browser to view the portfolio.
